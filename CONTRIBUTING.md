# Вклад в проект

Спасибо за интерес! 🎉

## Как помочь
1. Форкните репозиторий
2. Создайте ветку: `git checkout -b feature/моя-фича`
3. Внесите изменения, протестируйте на микросуммах
4. Commit: `git commit -m "feat: добавил X"`
5. Push и откройте Pull Request

## Правила
- Тестируйте транзакции только с суммами которые не боитесь потерять
- Не добавляйте внешние CDN — все зависимости локальные
- Не ломайте CSP (`connect-src 'self'`)
- Новые строки интерфейса добавляйте в **оба** языка в `lang.js`
